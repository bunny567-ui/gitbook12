# Ui Path

ReleaseOwl supports multiple credential types for registration. To register a **Ui Path** **Credential,** follow these steps:

1. Click on the **"Register Credential"** button.

<figure><img src="../../../.gitbook/assets/image (2) (1).png" alt=""><figcaption></figcaption></figure>

2. Fill out the credential form with the following details:

* **Credential Name**: Enter a reference name of your choice.
* **Credential Type**: Select **Ui Path**.
* **Scope** – Select the scope of the credential:
  * **Global** – Visible to all users.
  * **Private** – Visible only to the user who created it.
* **Token URL**: Enter the following URL: [https://account.uipath.com](https://account.uipath.com/).

<figure><img src="../../../.gitbook/assets/image (2031).png" alt=""><figcaption></figcaption></figure>

**To Obtain Client ID and Client Secret**:

1. Log in to your **UiPath Orchestrator** account.
2.  Click on your **profile icon** (usually in the top-right corner) and select **Preferences.**

    <img src="https://releaseowl.gitbook.io/releaseowl-docs/~gitbook/image?url=https%3A%2F%2F1890383800-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FDWyxe6hm5vqosFaByVgs%252Fuploads%252FuMYoLISBCiE4O2lHJN3v%252Fimage.png%3Falt%3Dmedia%26token%3Dab60391f-434a-4826-97c0-48ea7f110889&#x26;width=768&#x26;dpr=3&#x26;quality=100&#x26;sign=de84d476&#x26;sv=2" alt="" height="439" width="1429">
3.  In **Preferences**, go to the **Privacy & Security** section.

    <img src="https://releaseowl.gitbook.io/releaseowl-docs/~gitbook/image?url=https%3A%2F%2F1890383800-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FDWyxe6hm5vqosFaByVgs%252Fuploads%252FN3PReTZt7vm2UauCHcdv%252Fimage.png%3Falt%3Dmedia%26token%3D234ab58d-edbf-4d1d-a849-e613cb054111&#x26;width=768&#x26;dpr=3&#x26;quality=100&#x26;sign=26d6bf60&#x26;sv=2" alt="" height="687" width="1430">
4. Click on **View API Access**.
5. Here, you will find your **User Key**, which acts as both the **Client ID** and **Client Secret**.

<img src="https://releaseowl.gitbook.io/releaseowl-docs/~gitbook/image?url=https%3A%2F%2F1890383800-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FDWyxe6hm5vqosFaByVgs%252Fuploads%252FDlx9tQ1ujkX7OqMnXSxw%252Fimage.png%3Falt%3Dmedia%26token%3Da34a827e-d96a-4d14-b89b-5dd8423ed018&#x26;width=768&#x26;dpr=3&#x26;quality=100&#x26;sign=d190230&#x26;sv=2" alt="" height="624" width="1429">

6. **Copy and Paste Credentials**:

* Copy the **Client ID** and **Client Secret** from the **View API Access** section in UiPath.

<img src="https://releaseowl.gitbook.io/releaseowl-docs/~gitbook/image?url=https%3A%2F%2F1890383800-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FDWyxe6hm5vqosFaByVgs%252Fuploads%252FSm6ABVvFozMYnG0954i7%252Fimage.png%3Falt%3Dmedia%26token%3D7069f51c-4fe2-4e36-965c-7e4ae7d98941&#x26;width=768&#x26;dpr=3&#x26;quality=100&#x26;sign=c98abf0c&#x26;sv=2" alt="" height="836" width="680">

7. Return to the Register Credential page in ReleaseOwl.
8. Paste the Client ID and Client Secret into the appropriate fields.
9. Click **Save** to register the credential.
