# SAP Public Cloud

This guide explains how to manage and deploy SAP S/4HANA Public Cloud artifacts using **ReleaseOwl**. It covers the end-to-end workflow: setting up credentials and environments, assigning environments to projects, synchronizing transports and software collections, creating and configuring release pipelines (validation and deployment tasks), promoting user stories, and monitoring via validation reports and deployment logs. Follow the step-by-step sections below to ensure secure, traceable, and repeatable deployments.
