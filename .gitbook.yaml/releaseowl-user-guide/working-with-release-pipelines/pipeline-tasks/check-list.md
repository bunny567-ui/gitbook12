# Check List

A **Checklist Task** is used to perform **pre-release validation and compliance checks** before proceeding with deployment. It ensures that all required conditions, validations, and governance steps are completed as part of the release process.

<figure><img src="../../../.gitbook/assets/image (1760).png" alt=""><figcaption></figcaption></figure>

Fill in the required details:

| **Field**                      | **Description**                               | **Action to Perform**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| ------------------------------ | --------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Name**                       | Name of the checklist task                    | Enter a valid name using letters, numbers, underscores (\_) or periods (.) (e.g., `Check_List_SAS`)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **Description**                | Purpose of the checklist                      | Provide details such as _“Pre-release validation and compliance checks”_                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **Assign To**                  | User responsible for completing the checklist | <p>Select a user from the available list:                                      <br><strong>User:</strong><br>Select a specific user from the available list. The selected user will be responsible for completing the checklist task and ensuring all pre-release validation and compliance checks are fulfilled.<br><strong>Role:</strong><br>Any user with the selected role will be able to complete the checklist task. This is useful when checklist execution is not tied to a specific individual but to a functional role (e.g., QA, Release Manager).<br><strong>Custom:</strong><br>The task can be completed by any user associated with the <strong>custom-defined roles assigned to each component</strong>. This provides flexibility to assign different roles for different components involved in the checklist validation process.      <br><strong>Promoter:</strong><br>The assigned user/role/custom-mapped user acts as the promoter who completes all checklist items.  </p> |
| **Disable Email Notification** | Control email alerts                          | Enable to stop email notifications for this task.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **Approval Message Required**  | Enforce comments during completion            | Enable to make comments mandatory while completing the checklist                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **Checklist Items**            | Define validation steps                       | Click **+ Add** to create checklist items                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **Checklist Item Name**        | Name of each validation step                  | Enter clear validation or compliance step names                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **Mandatory**                  | Indicates compulsory checks                   | Enable to ensure the step must be completed before proceeding                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **Evidence Required**          | Proof for compliance                          | Enable to require supporting documents/screenshots as evidence                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
